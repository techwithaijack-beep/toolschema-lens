# ToolSchema Lens launch kit

## One-line positioning
ToolSchema Lens helps AI engineers spot bloated tool schemas before those contracts quietly destroy prompt budget and tool-call reliability.

## Why it matters now
Agent stacks are shipping more tools, bigger MCP contracts, and heavier JSON Schema metadata. Teams notice cost and instability, but often miss the hidden culprit: oversized tool contracts repeated every turn.

## Demo script
1. Run the sample analyzer command.
2. Open the generated HTML report.
3. Show before/after token estimates and the optimized contract.
4. Point out large enums, duplicated descriptions, and runtime-irrelevant metadata.
